Documentation
=============

x_epi can be used three ways:

#. Directly calling the :doc:`XEPI <api>` Python class
#. A simple :doc:`graphical user interace <ui>`
#. A :doc:`command line wrapper <cmd>`

I would recommend starting out with either GUI or command line interface. Please see
the  documentation page for each method for more information.
