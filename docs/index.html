x_epi is a Python library for creating echo-planar imaging (EPI) sequences for X-nuclei. It uses the Pulseq framework implemented in PyPulseq.

Users can create custom EPI sequences using several different methods. Multiple features are available including:

  1. Spectral Spatial Radio-frequency (SSRF) excitation

  2. Symmetric or flyback readouts with an arbitrary number of echoes

  3. 2D or 3D imaging

  4. Partial Fourier in either phase encoding direction

  5. Optional ramp sampling

Although this package was designed for imaging hyperpolarized 13C agents, it can be used for several different nuclei. Please feel free to ask if you have any questions!
